## 📌 Project Overview

This project is an automated job aggregation system built using n8n. It collects job listings from multiple platforms such as Remotive and WeWorkRemotely, processes and filters them using JavaScript, and exports structured data into Excel.

## ⚙️ Features

- Automated job scraping from multiple sources
- Data cleaning and transformation using JavaScript
- Merging and deduplication of job listings
- Scheduled execution (daily automation)
- Export to XLSX file
- Scalable workflow design

## 🛠️ Tech Stack

- n8n (Workflow Automation)
- JavaScript (Data Processing)
- REST APIs
- Docker (optional)
- Data Cleaning & Transformation

## 🚀 Future Improvements

- Integration with Telegram/Email notifications
- Storing data in database (PostgreSQL)
- AI-based job relevance scoring
